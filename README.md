# Terminal Civilization Engine

## Overview
A turn-based civilization simulation engine built in Python.
The program generates a procedural world map and simulates AI factions expanding, gathering resources, and battling for territory.

## Goals
- Demonstrate system design
- Practice algorithms and data structures
- Implement a clean modular architecture
- Follow professional Git workflow

## Features (Planned)
- Procedural terrain generation
- AI-controlled factions
- Resource economy system
- Turn-based simulation loop
- Combat resolution system
- Territory expansion tracking

## How to Clone

git clone https://github.com/ShiriezH/terminal-civ-engine.git

## How to Run

cd terminal-civ-engine  
python main.py

## Project Structure

world/ → Map and terrain generation  
entities/ → Factions and units  
systems/ → Movement, combat, resources  
utils/ → Display helpers  
tests/ → Testing modules
